# PUG+SASS PREPROCESSOR

## 🤨 Навіщо ?:
щоб на цій основі можна було швидко і зручн писати на SASS+PUG із конвертацією в чистий HTML+CSS

## 🤓 Опис:
службовий скрипт. заготовка для написання сайтів на PUG+SASS і конвертації цього кода в чистий HTML+CSS

## ☠️ Використані технології:
- все написано на PYTHON
- GUI на KIVY
- під капотом працює на AMPY

## 🌱 Структура проекта:
- `src/` — тут можна писати код, папки вже підготовані
- `gulpfile.js` - системний файл
- `package-lock.json` — системний файл
- `package.json` — системний файл
